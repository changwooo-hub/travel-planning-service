# 🧳 Travel Planning Service

관광지와 숙소 데이터를 통합하고, AI가 자동으로 여행 코스를 추천해주는 통합 여행 플래닝 서비스입니다.

---

## 📌 프로젝트 개요

기존 여행 서비스가 제공하지 못하는 '개인 맞춤형 코스 생성' 기능을 제공하고, 관광지·숙소 데이터를 통합해 계획-코스-리뷰까지 한 플랫폼에서 처리할 수 있도록 개발했습니다.

- **개발 기간**: 2025 (부트캠프 세미 프로젝트)
- **팀 구성**: 4인 (김창우, 김하연, 이승아, 최민혁)
- **담당 역할**: 사용자 인증 시스템 및 관리자 데이터 관리 기능

---

## ✨ 주요 기능

- **여행 계획 관리**: 기간 기반 Day 자동 생성, 날짜 변경 시 Day 재정렬
- **코스 관리**: Day별 코스 구성, Drag & Drop 순서 재정렬, AI 추천 코스 연동
- **AI 코스 생성**: 사용자 피드백 기반 재추천 기능 포함
- **리뷰 시스템**: 관광지/코스 리뷰 작성, 별점, 이미지 첨부
- **관광지/숙소 정보**: 공공 API 연동, 지도 기반 위치 시각화, 거리순 정렬
- **관리자 대시보드**: 관광지/숙소/회원 데이터 CRUD, 소프트 삭제

---

## 🙋‍♂️ 담당 파트 (김창우)

### 사용자 인증 시스템
- JWT 기반 Access Token / Refresh Token 이중 구조 설계
- HttpOnly 쿠키로 토큰 관리하여 XSS 공격 방어
- 401 응답 시 Refresh Token으로 Access Token 자동 갱신 (Race Condition 방지 처리)
- 로그아웃 시 Refresh Token DB 삭제로 세션 무효화
- 비밀번호 해싱 및 로그인/회원가입 API 개발

### 권한 관리 (RBAC)
- USER / ADMIN 역할 기반 접근 제어
- 권한 검증 커스텀 어노테이션 구현
- 관광지 추가/수정/삭제는 ADMIN 권한만 허용하도록 설계

### 관리자 데이터 관리
- 관리자 대시보드에서 회원/관광지/숙소 데이터 CRUD 기능 구현
- 관광지 삭제는 활성/비활성 처리로 소프트 삭제 구현
- 데이터 수량 및 상세 조회 기능 구현

---

## 🛠 기술 스택

**Backend**
- Spring Boot, JPA/Hibernate
- REST API
- JWT, Spring Security

**Data**
- 공공 Open API (한국관광공사)
- Travel DB 자체 구축 (관광지 12,900개, 숙소 3,500개)

**Frontend**
- 지도 기반 UI (카카오맵 API)
- 반응형 디자인

---

## 🏗 핵심 엔티티

- **User**: 사용자 정보 관리
- **TripPlan**: 여행 계획 정보
- **Course / CourseItem**: 코스 및 코스 내 항목
- **TourSpot / Accommodation**: 관광지 및 숙소 정보
